## Whisper

### Instalacion Whisper
El código base tiene una compatibilidad con las versiones de Python 3.8 - 3.11.
Para instalar o actualizar la última versión de Whisper se realiza mediante el comando:
  ```bash
  pip install -U openai-whisper
```
